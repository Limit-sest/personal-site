MHD Run je dynamická městská hra, která kombinuje závod, veřejnou dopravu a real-time strategii. Hra je inspirovaná populárním zahraničním formátem _Jetlag: The Game_, který jsem adaptoval pro české prostředí.

Vytvářel jsem vlastní herní mechaniky, úkoly a pravidla. Také jsem vytvořil progresivní webovou aplikaci ve Vue.js pro jednodušší průběh hry. Díky ní si hráči nemusí pamatovat, kolik mají herních peněz, jaké karty si vylosovali, ani kde se nachází jejich cíl.
